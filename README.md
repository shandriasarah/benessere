# Benessere

Sistema de gestão para salões de beleza e estética, voltado ao gerenciamento de agendamentos, clientes e serviços. O nome deriva do italiano *benessere* ("bem-estar"), refletindo a proposta de simplificar a rotina operacional de negócios do setor de beleza e autocuidado.

**Demo:** [benessere-ssg3.vercel.app](https://benessere-ssg3.vercel.app)

## Visão geral

Aplicação full stack desenvolvida para centralizar a gestão de um salão de beleza. O sistema permite o controle de agendamentos, o cadastro e acompanhamento de clientes e a administração dos serviços oferecidos, com autenticação de acesso e interface responsiva.

## Funcionalidades

- Agendamento e gerenciamento de horários
- Cadastro e histórico de clientes
- Catálogo e controle de serviços
- Autenticação de usuários

## Instalação

Pré-requisitos: Node.js instalado.

```bash
# Clonar o repositório
git clone https://github.com/shandriasarah/benessere.git

# Acessar o diretório
cd benessere

# Instalar as dependências
npm install

# Executar o projeto
npm start
```
